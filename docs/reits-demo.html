5层漏斗筛选 : 分红率 → 收入趋势 → 流动性 → AI舆情 → AI综合评选
82只C-REITs全覆盖 → 5层逐步过滤 → 最终推荐Top 5；三模型(MiniMax M2.5+GLM-5+Kimi K2.5)投票决策；每周日凌晨1点自动运行
第1-2层: 基本面筛选 iFinD
基于iFinD数据，从分红率和收入趋势两个维度进行硬性筛选
分红率筛选 第1层
硬性范围3-10%，优选5-8%区间。分红率过低说明底层资产收益不佳，过高可能暗示价格暴跌
三级降级策略 数据源
① iFinD基金指标 → ② 日频序列 → ③ 历史行情除权缺口估算
收入环比剔除 第2层
取最近4个季度收入数据，若≥75%季度环比下降则剔除
第3层: 流动性筛选 iFinD
基于iFinD历史行情数据，8线程并发获取，筛除流动性不足的品种
零换手率剔除 核心
检查最近10个交易日的成交量，全部为0则剔除
市场数据提取 增强
同时提取日均换手率、5/20日涨跌幅、30日价格区间等指标
第4层: AI舆情分析
三大模型并发分析REITs负面舆情，投票决策
负面事件检测 5类
底层资产问题 · 管理人违规 · 分配下降 · 诉讼查封 · 重大利空
三模型投票 机制
MiniMax M2.5 + GLM-5 + Kimi K2.5各自独立判断，超半数认为负面才剔除
保守策略 原则
宁可错留、不可错杀。单一模型幻觉不会导致误判
第5层: AI综合评选
三模型独立推荐+加权融合+共识加分，输出最终Top 5
5维选择标准 权重
分红率稳健30% · 类型多元化25% · 流动性15% · 近期走势15% · 底层资产质量15%
加权融合评分 算法
模型评分×75% + 排名加分×25%，按MiniMax(40%)/GLM(30%)/Kimi(30%)加权
共识加分 机制
2个模型同时推荐+10%，3个模型全票推荐+20%
📈 回测评价机制
每期推荐自动计算1/3/6个月收益率（推荐日买入收盘价，目标日最近交易日收盘价）。三模型联合生成回测评价，从收益表现、稳定性、持有期特征和改进建议四个维度进行分析。
🛡️ 降级保障机制
数据降级：分红率获取支持三级降级（iFinD指标→日频序列→历史行情估算），确保数据可用性。
AI降级：若所有模型调用失败，自动切换为分红率+类型多元化的量化排序选择，保证推荐输出。
筛选降级：各层数据不可用时自动跳过（全部保留），不会因数据缺失导致误杀。
